הבדל ראשון: המטרה לשמה נבנה כל web

  • web 1.0: מקשר בין מידע לאנשים.
  • web 2.0: מקשר בין אנשים (קיימת אינטרקציה בין אנשים ברשת. רשתות חברתיות וכו').
  • web 3.0: מקשר ידע. שימוש בלמידת מכונה כדי להסיק מסקנות ולעזור בתהליך קבלת ההחלטות.

    • הבדל שני: הפעולות אותן יכול המשתמש לבצע

  • web 1.0: קריאה בלבד. רק יוצר האתר יכול לכתוב תוכן.
  • web 2.0: קריאה וכתיבה. גם יוצר האתר וגם המשתמשים יכולים לכתוב תוכן.
  • web 3.0: כתיבה, קריאה והרצה. קיימים דפים אינטרקטיביים.

    • הבדל שלישי: אופן החיפוש

  • web 1.0: מנועי החיפוש השתמשו במאקרו תוכן. דבר שגרם לחיפוש להיות מהיר אך לא בהכרח רלוונטי.
  • web 2.0: מנועי החיפוש החלו להשתמש במיקרו תוכן, תגיות. יוצרי האתר יכלו להזין תגיות באופן ידני.
  • web 3.0: מנועי החיפוש החלו לתייג מידע באופן אוטומטי.

    • תרחיש שימוש עבור השאילתא - "מתכון לשקשוקה":

    web 1.0:

  • יחזרו דפים המכילים את המילים "מתכון" ו"שקשוקה".
  • + החיפוש יהיה מהיר.
  • - תוצאות החיפוש יהיו זהות לכל המשתמשים שיחפשו את השאילתא הזאת (החיפוש לא מותאם אישית).
  • - יכולות לחזור תוצאות לא רלוונטיות. לדוגמא: תוצאות שמכילות מתכונים לא של שקשוקה או תוצאות שקשורות לשקשוקה אך לא כוללות את המתכון.

    • web 2.0:

  • דפי האינטרנט מכילים תגיות שנוצרו על ידי יוצרי הדף.
  • + התוצאות יכילו דפים שבעלי האתר תייגו כרלוונטים לשאילתא.
  • - יוצרי הדף יכולים להזין תגיות שלא משקפות את תוכן האתר ובכך לפגוע ברלוונטיות התוצאות שהמנוע מחזיר.

    • web 3.0:

  • דפי האינטרנט מכילים דפים שתוייגו באופן אוטומטי. יש שימוש בלמידת מכונה כדי לספק תוצאות יותר רלוונטיות.
  • + מנוע החיפוש יתאים את התוצאות למשתמש ולכן סבירות גבוה יותר לקבל תוצאות רלוונטיות.
  • + תיקון שגיאות כתיב.
  • + הצעות להשלמה אוטומטית של השאילתא.
